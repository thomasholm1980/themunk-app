# Product Principles

## What The Munk is

- An interpretation engine for physiological self-regulation
- A ritual interface, not a productivity tool
- A system that builds pattern awareness over time

## What The Munk is not

- Not a dashboard
- Not a metrics display
- Not a medical device
- Not a diagnostic tool
- Not an optimization theater

## Core UX philosophy

- Calm > stimulation
- Interpretation > reporting
- Awareness > action pressure
- Stable rendering > animation experiments
- Glanceable meaning > data density

## Language guardrails

- No diagnostic terms
- No medication references
- No overconfident assertions
- Enforce hedged language: "may", "could", "correlates", "suggests"
